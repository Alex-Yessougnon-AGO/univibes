# Message Transmutation Reference

> A winning sales message should travel across formats. The point is not more ad variants; it is the same psychological hook expressed through formats that create different signals for Meta and different experiences for the buyer.

## Core Formats

### 1. AI UGC / First-Person VSSL

Use when:

- cold traffic needs empathy and belief installation;
- the prospect believes their problem is hard or unique;
- the offer can be framed as "I was just like you, but worse."

Framework:

1. **I was just like you, but worse:** narrator starts from a more severe version of the prospect's problem.
2. **Discrediting hook:** undermine the old explanation or common advice.
3. **Without bridge:** sell the desired outcome without the hated alternative.
4. **Mechanism reveal:** name the Unique Mechanism that made the change possible.
5. **Proof and next action:** short proof point, then CTA.

Example shape:

```text
My doctor said it had nothing to do with [old explanation].
But [specific symptom] kept happening.
What changed was [Unique Mechanism], not [common alternative].
[Proof]. [CTA].
```

Do not impersonate a real person or fabricate medical/financial claims. Use first-person only when the brand has a real narrator, customer quote, creator script, or clearly marked concept.

### 2. Native Static / Scroll Stopper

Use when:

- speed-to-data matters;
- creative needs to blend into feed but break the vertical's pattern;
- primary text carries most of the selling.

Copy implication:

- primary text must do the argument;
- headline should be concrete and short;
- visual note should specify the contrast move, even if this skill only writes copy.

### 3. AI Animation / Look-Back Strategy

Use when:

- the concept benefits from simple visualization or metaphor;
- realism is not the point;
- older direct-response TV/commercial structures can make the mechanism memorable.

Copy implication:

- write a simple visual sequence;
- avoid overexplaining;
- connect animation to the mechanism and proof.

## Advertorial Pre-Lander Pattern

For cold traffic where direct-to-product is too abrupt, recommend an advertorial pre-lander.

Flow:

1. **Native ad:** emotional/story hook.
2. **Advertorial:** hybrid editorial page that installs beliefs and objections before price.
3. **Sales page/PDP:** product details and purchase/trial action.
4. **Order/signup page:** conversion.

Ad-copy output should not write the full advertorial unless asked. It should include an `Advertorial Pre-Lander Brief` when the strategist chooses this pattern:

- belief sequence to install,
- old way to discredit,
- Unique Mechanism to introduce,
- proof required,
- CTA from advertorial to sales page.

### Chad Funnel Variant

Use the Chad Funnel variant when the offer needs a native-ad-to-editorial bridge before the product page can convert.

Flow:

1. **Native Meta ad:** creates curiosity and starts one belief shift.
2. **Advertorial:** discredits the old belief, installs the remaining beliefs, introduces the Unique Mechanism, and substantiates the proof.
3. **Product page / PDP / sales page:** shows the offer, details, pricing, and risk reversal.
4. **Order or signup page:** completes purchase, trial start, lead capture, or install.

The ad-copy skill should not write the whole funnel unless explicitly asked. It should produce the ad plus an `Advertorial Pre-Lander Brief` that names the belief sequence and handoff requirements for the downstream page.

### 6 Necessary Beliefs Bridge

For advertorial and Chad Funnel variants, identify 3-6 beliefs that must be installed before the prospect sees the price or product page:

1. The problem is real and costly in the prospect's current context.
2. The common explanation is incomplete or wrong.
3. The desired outcome is possible for someone like them.
4. The old alternatives fail for a specific reason.
5. The offer's Unique Mechanism changes the constraint.
6. The next step is timely and low-risk.

Short Meta ad copy should usually install only one belief and point to the advertorial. If it tries to install all six, it will either exceed the visible window or become generic.

## Variable Subtraction

When results are ambiguous, change one variable at a time:

| Symptom | Likely constraint | Next isolated test |
|---|---|---|
| Good CTR, low conversion | Funnel/offer constraint | Keep winning creative, test landing page or offer. |
| High CPC, good conversion | Ad/creative constraint | Keep funnel fixed, test hook/format/creative angle. |
| Low CTR and low conversion | Message-market constraint | Test a new sales message, not just a new visual. |
| Good early metrics, fast fatigue | Creative-volume constraint | Transmute winning message across formats. |

Strategist should name the one variable each variant tests. Composer should avoid changing hook, format, proof, CTA, and offer all at once unless the brief explicitly asks for net-new concepts.
